Dice Roll API

This API generates random dice rolls for any combination of dice.

Built as a test to try out Now

How it works

Simply add the amount and type of dice you want to the URL.

Examples:
/2d6 will roll two 6-sided dice.
/4d12 will roll four 12-sided dice.
/d4 will roll one 4-sided die.

Results are returned in JSON format by default, but you can add the extension .xml to any roll to get an XML structure instead.
